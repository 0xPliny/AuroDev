## Overview

This document provides an index of shared libraries used in the Auro System.

---

## Constants and Definitions

### VB.NET Constants
- [ICISDefines.vb](ICISDefines.md) - **Complete VB.NET Constants Reference**
  - System type constants, date/time formats
  - Communication classes (COM status)
  - Device status enumerations (DEV, ESTAT)
  - Station types (36) and modes (14)
  - Move status constants (22 statuses)
  - Vehicle functions, schedule states, completion types
  - Stacker functions (34)
  - Location constants and system maximums
  - Station error codes (30+)

### C++ Global Parameters
- [global_prm.h](global_prm.md) - **Complete C++ Constants Reference**
  - Type definitions (GP_BOOL, GP_UINT, etc.)
  - Return codes (GP.GOOD, GP.BAD, etc.)
  - Log level constants (0-10)
  - Project-specific maximums
  - Macro definitions, control characters
  - PLC definitions, file paths
  - GP class reference (VEHICLE, SR, ST, MOVS)

**Important:** ICISDefines.vb and global_prm.h must be kept synchronized.

  - ds_sql - SQL interface

  - ds_invt - Inventory database functions

## Operation Subroutines (osub/)

- [OSUB](osub.md) - **Auto-Generated Database Access Routines**
  - Generated by `p_ut_makdb` from XML table definitions
  - Standardized CRUD operations for all database tables
  - Functions: select, insert, update, delete, count, sum
  - Table structures: SYS, MOVS, STK, LOAD, LOCATION, INVENTORY
  - **Note:** Do not modify directly - regenerate from XML

---

## Common Control Subroutines (ccsub/)

- [CCSUB](ccsub.md) - **C++ Equipment Control Classes**
  - `cc_stk` - Stacker crane control
  - `cc_vehicle` - AGV/RTNX vehicle control
  - `cc_mudp` - MUDP protocol communication
  - `cc_plc` - PLC communication abstraction
  - `cc_str` - String manipulation utilities
  - `cc_sys` - System-level control
  - `cc_mem` - Mapped memory management

---

## Common Subroutines (csub/)

- [CSUB](csub.md) - **Fundamental Utility Functions**
  - `cs_log_printf` - Formatted logging with levels
  - `cs_msg_put/get/peek` - Message queue operations
  - `cs_reg_read/write` - Registry access
  - `cs_dtm_*` - Date/time manipulation
  - `cs_err` - Error handling utilities
